
下载
~~~git
git clone https://github.com/GreaterWMS/GreaterWMS-Doc-CN.git
~~~

安装
~~~python
pip install -r requirements.txt
~~~

启动
~~~python
mkdocs serve
~~~

局域网启动
~~~shell
mkdocs serve --dev-addr 0.0.0.0:8009
~~~

MarkDown摆放位置
~~~shell
material/overrides
~~~

图片位置
~~~shell
docs/assets/images/
~~~
